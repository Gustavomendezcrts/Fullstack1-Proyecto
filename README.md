# FilmFlux
## Modelos
>[!NOTE]
>Pelicula
>- Id
>- IMDb
>- Titulo
>- Director
>- Fecha de estreno
>- Genero
>- Puntaje (1 - 10)


>[!NOTE]
>Serie
>- Id
>- IMDb
>- Titulo
>- Director
>- Fecha de estreno
>- Genero
>- Puntaje (1 - 10)
>- Temporadas (Cantidad)

>[!NOTE]
>Usuario
>- Nd
>- Nombre
>- Email
>- Contraseña
>- Gusto Principal (serie/pelicula)
>- Gusto Secundario (serie/pelicula)

## BD's
>[!NOTE]
>La base de datos con nombre FilmFlux se crea de forma automatica con el mysql de Laragon en el caso de que no se encuentre la base de datos
