Pelicula

- IMDb
- Titulo
- Director
- Fecha de estreno
- Genero
- Puntaje (1 - 10)

Serie
- Id
- IMDb
- Titulo
- Director
- Fecha de estreno
- Genero
- Puntaje (1 - 10)
- Temporadas (Cantidad)

Usuario
- id
- nombre
- email
- contraseña
- Gusto Principal (series/peliculas)
- Gusto Secundario (series/peliculas)
