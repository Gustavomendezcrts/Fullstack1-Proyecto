# FilmFlux
## Modelos
>**Pelicula**
>- Id
>- IMDb
>- Titulo
>- Director
>- Fecha de estreno
>- Genero
>- Puntaje (1 - 10)


>**Serie**
>- Id
>- IMDb
>- Titulo
>- Director
>- Fecha de estreno
>- Genero
>- Puntaje (1 - 10)
>- Temporadas (Cantidad)

>**Usuario**
>- Nd
>- Nombre
>- Email
>- Contraseña
>- Gusto Principal (serie/pelicula)
>- Gusto Secundario (serie/pelicula)
